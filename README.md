# Temple Quest: French Vocab (School & Home)

A tiny GitHub Pages game to revise French vocab with typing + sound effects.

## Run locally
Just open `index.html` in your browser.

## Put it on GitHub Pages
1. Repo → Settings → Pages
2. Source: **Deploy from a branch**
3. Branch: **main**, folder: **/** (root)
4. Save — your link appears at the top.

## How to play
- Choose a mode:
  - **English ➜ French** (teacher says English, you type French)
  - **French ➜ English** (teacher says French, you type English)
- Accents are optional.
- Use Hint/Reveal/Skip if you get stuck.
